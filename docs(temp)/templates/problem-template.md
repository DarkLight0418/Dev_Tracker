# Problem: {문제 한 줄 요약}

## 1. 문제 발생 배경

- 언제 발생했는가?
- 어떤 작업을 하던 중이었는가?
- 정상적으로 기대한 동작은 무엇이었는가?

## 2. 실제로 발생한 현상

- 눈에 보인 증상은 무엇이었는가?
- 에러 메시지 / 로그 / 화면 변화
- 재현 가능한가? (예 / 아니오)

## 3. 1차 가설

- 처음에 의심한 원인은 무엇이었는가?
- 왜 그렇게 생각했는가?

## 4. 확인 및 시도

- 시도한 해결 방법들
- 각 시도의 결과
- 실패한 이유 (추정)

## 5. 근본 원인

- 최종적으로 확인된 원인
- 구조적 / 환경적 / 코드적 원인

## 6. 해결 방법

- 실제로 적용한 해결책
- 핵심 변경 사항
- 주의할 점

## 7. 검증

- 문제가 해결되었는가?
- 어떤 기준으로 확인했는가?
- 부작용은 없는가?

## 8. 회고

- 이 문제에서 놓쳤던 신호
- 다음에 같은 문제를 줄이기 위한 기준
