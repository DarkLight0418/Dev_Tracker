# Decision Log – 로그인 이슈

---

## 2025-12-21

### 1. 문제 상황 인식
- 새로고침 시 로그인 상태가 유지되지 않음
- 로그인 API 리팩토링 직후부터 발생

---

### 2. 당시 전제 조건
- 로그인 로직 자체는 정상 동작 중
- 콘솔 에러 없음
- 서버 로그에 명확한 예외 없음

---

### 3. 초기 가설
- 세션 또는 서버 인증 로직 문제일 가능성이 높다고 판단

> 판단 근거  
> - 과거에 유사 증상을 서버 세션 설정으로 해결한 경험이 있음  
> - API 리팩토링 직후 발생한 문제라는 맥락

---

### 4. 선택한 접근
- 서버 인증 흐름부터 점검하기로 결정
- 브라우저 정책(SameSite, Secure 등)은 후순위로 둠

---

### 5. 실행 결과
- 서버 로직상 명확한 오류 발견되지 않음
- 세션 생성/유지 로직 정상 확인

---

### 6. 판단 수정 포인트
- 서버 측 이상 징후가 없다는 점에서  
  클라이언트 환경 또는 브라우저 정책 문제 가능성 증가

---

### 7. 다음 판단 방향
- HTTPS 여부 및 쿠키 옵션 확인 필요
- 브라우저 정책 관련 설정 점검 예정
